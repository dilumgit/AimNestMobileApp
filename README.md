# AimNest Mobile App

AimNest is a mobile financial management application developed using Kotlin in Android Studio. The application helps users manage income, expenses, savings targets, financial analysis, and smart financial recommendations efficiently.

## Features
- Profile management
- Income management
- Expense management
- Financial target management
- Smart financial suggestions
- Alerts and warnings
- Notification center
- Monthly income summary
- Overall financial analysis
- Premium subscription functionality

## Technologies Used
- Kotlin
- Android Studio
- SharedPreferences
- Material Design Components
